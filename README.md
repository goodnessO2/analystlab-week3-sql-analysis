# AnalystLab Africa Week 3: SQL & Data Querying

This project was completed as part of the AnalystLab Africa Data Analytics Internship Program.

## Project Overview

The goal of this task was to practice SQL querying and business analysis using two datasets:

1. Chinook Database - digital music store data
2. Sample Sales Dataset - sales transaction data

## Tools Used

- MySQL Workbench
- SQL
- Google Docs / Microsoft Word

## SQL Concepts Applied

- SELECT, WHERE, ORDER BY
- GROUP BY and HAVING
- Aggregate functions: SUM, AVG, COUNT
- INNER JOIN
- Subqueries
- Common Table Expressions
- Window functions: RANK, ROW_NUMBER, PARTITION BY
- Indexing concepts for query optimization

## Key Insights

- The Chinook store generated $2,328.60 in revenue from 412 invoices.
- USA was the highest revenue-generating country in both datasets.
- Rock was the top-performing genre in the Chinook Database.
- Jane Peacock was the top-performing Chinook sales support agent.
- The Sales Dataset generated total sales of $10,032,628.85.
- Classic Cars was the best-performing product line.
- Euro Shopping Channel was the highest revenue-generating customer.
- November 2004 recorded the highest monthly sales in the Sales Dataset.
